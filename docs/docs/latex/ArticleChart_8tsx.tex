\doxysubsection{Detailed Description}
Component for displaying stock price charts with predictions. 

This component renders a line chart showing\+:
\begin{DoxyItemize}
\item Historical stock prices
\item Model predictions
\item Optional LSTM predictions Uses Chart.\+js for rendering 
\end{DoxyItemize}
